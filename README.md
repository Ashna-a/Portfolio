## üöÄ Local Development

To run this portfolio locally:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Ashna-a/Portfolio.git](https://github.com/Ashna-a/Portfolio.git)
    cd Portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm start
    ```
